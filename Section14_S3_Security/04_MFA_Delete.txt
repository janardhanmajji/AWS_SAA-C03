MFA -Multi Factor AUthentication

MFA IN S3 -> Multi Factor AUthentication in s3 forces the users to send a code on a device before doing important operations on S3 

Ex: Google AUthenticator etc..

MFA Required when:
    *Permanantly delete an object version
    *Suspend versioning on bucket

MFA wont Required when:
    *Enable versioning
    *List deleted versions

-> To use MFA Delete versioning must be enabled on the bucket.
-> Only the bucket owner(root account) can enable/disable MFA delete
