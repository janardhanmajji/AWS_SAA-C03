PreSigned URL's

-> They can be generated using Console, CLI or SDK
-> They can be generated by the owner of the bucket which any user can access for a limited amount of time.

-> URL Expiration:
    *Console - (upto 12 hours)
    *AWS CLI - (upto 168 hours)
-> They can be used for GET/PUT (upload and download)

Ex: *Allow logged in user to download a private video
    *Allow temporarily a user to download files to a precise location in S3 Bucket.
